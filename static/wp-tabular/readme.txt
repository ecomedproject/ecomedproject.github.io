== Description ==

WordPress Tabular Plugin is a very easy to Table Builder Plugin for WordPress. WordPress Tabular Plugin has Complete GUI for creating and sorting rows and columns without writing any html. Also WordPress Tabular Plugin packed with many css table templates, where you can select any template to show a table at front-end. As well as regular short-code, It will provide you js embed and you can use the js code even in another site to embed the table. Tables are exportable in xml, html and csv format. Also This WordPress Table Builder Plugin 

= Features =
* Easy administration
* Drag and Drop option for re-ordering columns
* Drag and Drop option for re-ordering rows
* Short-code generator - Create unlimited short-code and place whatever you like there
* Integrated Table template css
* Tinymce button for inserting table short-code with selecting table and table template



== Installation ==

1. Upload `wp-tabular` to the `/wp-content/plugins/`  directory
2. Activate the plugin through the 'Plugins' menu in WordPress
